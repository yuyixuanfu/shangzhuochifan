# 出门买菜上桌吃饭

一个给 AI 玩的文字买菜做饭游戏。

你不是厨师。你是一个每天想"今晚吃什么"的普通人——拿着十几块钱，穿过阴晴雨雪，挤进嘈杂的菜市场，跟摊主讨价还价，挑一把带虫眼的青菜，回家开火，把一顿饭端上桌。

**一局 = 一顿饭**：领预算 → 出门 → 逛菜场 → 买 → 回家 → 做菜 → 端上桌 → 她吃了说什么

---

## 这游戏玩什么

你是一个有伴侣的普通人。每天早上领到一笔买菜钱（8~28元，够紧的），出门走到菜市场，在有限时间里买到尽可能好的食材，回家做饭，端上桌。

菜场不是超市——摊主有自己的性格和心情，菜有好有坏有暗坑，砍价看人情，天气影响品种和价格，时间不够就得赶。做菜不是点按钮——你真的要一步步写：切番茄、热锅、倒油、炒蛋、盛出、再炒番茄、放回去、加盐、出锅。做得好她夸你，做砸了她也吃。

**她不是 NPC**——你的伴侣是真人。游戏不编她的口味、不替她反应。她说好吃就是好吃，她说太咸你就得记着下次少放盐。

## 游戏流程详解

### 一局的结构

```
领预算 → 出门路途 → 逛菜场（限时）→ 回家 → 做菜 → 端上桌 → 她说
  ↑                                                    ↓
  └────── 新一天，冰箱里的剩菜可能过期 ←────────────────┘
```

### 领预算

每天早上随机领 8~28 元。紧巴巴的——买好菜就得砍价，不砍价就得多走几家比价。

**攒钱罐**：昨天没花完的钱，一半结转到今天预算，另一半进攒钱罐。攒钱罐跨天累积，攒着买大件（排骨/螃蟹）。用「取罐」取出加到今天的菜钱。

### 出门路途

根据天气不同，出门的体验不同：
- 晴天：溜达过去，袋子晃来晃去
- 阴天：风往领口里钻
- 雨天：鞋踩进水坑，袋子勒手指

### 逛菜场

菜场有 **16 个固定摊位**，分 6 个区：

| 分区 | 摊位 | 卖什么 |
|------|------|--------|
| 蔬菜区 | 王姐蔬菜、老张菜摊、小陈鲜蔬 | 小白菜、菠菜、生菜、茼蒿… |
| 根茎区 | 刘姐根茎、赵哥山货 | 土豆、萝卜、山药、春笋… |
| 瓜果区 | 孙姐瓜果、阿亮果蔬 | 番茄、黄瓜、茄子、青椒… |
| 豆菌区 | 周婶豆角、老吴菌菇 | 豆角、香菇、平菇、金针菇… |
| 肉鱼区 | 胖哥肉铺、李姐鲜肉、老何鱼摊、小黄水产 | 五花肉、排骨、鲫鱼、虾… |
| 蛋品区 | 马婶蛋铺 | 鸡蛋、鸭蛋… |
| 干货区 | 老方干货 | 虾皮、紫菜、干辣椒… |

还有 **3 个流动摊**（随机出现）：
- 老农自种菜（便宜但量少）
- 散市尾货（快收摊了，甩卖）
- 时令限定（按季节卖香椿、荠菜、莲蓬、板栗等别处没有的）

### 时间系统

菜场有营业时间，不同时段时间点数不同：

| 时段 | 时间点数 | 特点 |
|------|---------|------|
| 早市 | 8点 | 货全但人多 |
| 上午 | 5点 | 正常 |
| 散市 | 3点 | 便宜但缺货多 |

每个操作消耗时间：逛摊 -1、砍价 -1、聊天 -1、细看 -0.5。时间归零强制回家。

### 买与砍价

看到菜可以直接买。价格受季节、天气、时段影响——当季便宜，反季贵；雨天涨价；散市甩卖。

**砍价**：`砍价 菜名 话术`——话术影响成功率：
- "老客户了" → 熟客加分
- "隔壁更便宜" → 竞争策略，有风险
- "多买点便宜点" → 量大策略

摊主 5 种性格影响砍价：
| 性格 | 砍价难度 |
|------|---------|
| 实在 | 容易，价格本来就实 |
| 爽快 | 一般 |
| 话唠 | 聊着聊着就便宜了 |
| 算计 | 难，算得比你精 |
| 死硬 | 最难，一分不让 |

### 品质与暗坑

菜的品质分五档：

| 品质 | 标记 | 说明 |
|------|------|------|
| 优品 | ★优 | 水灵、新鲜 |
| 不错 | ○好 | 正常水平 |
| 一般 | ～般 | 凑合能用 |
| 不太行 | ✗差 | 发蔫、有斑点 |
| 暗坑 | ！坑 | **表面看不出来，切/洗时才暴露** |

暗坑是核心机制——你看不出菜有问题，买回家切开才发现烂心、注水、里面发黑。细看（`细看 菜名`）可以提前发现，但消耗时间。

### 熟客系统

反复去同一个摊，熟客等级会提升：

| 等级 | 条件 | 权益 |
|------|------|------|
| 1级 面熟 | 3次 | 摊主认识你 |
| 2级 常来 | 5次 | 报价不虚高 |
| 3级 老客 | 8次 | 偶尔多送一点 |
| 4级 赊账 | 12次 | 钱不够也能买 |

### 天气与灾害

每天随机天气（晴/阴/雨），影响：
- 雨天：涨价 10%、少带 1 样菜、少 1 点时间、摊位可能关门
- 5 种随机灾害：

| 灾害 | 触发条件 | 效果 |
|------|---------|------|
| 暴雨淹市 | 雨天 | 蔬菜/根茎/瓜果区关闭，剩下的涨价 |
| 检查来了 | 任意 | 时间 -3 |
| 摊主缺席 | 任意 | 随机关 2 家 |
| 批发涨价 | 任意 | 全面涨价 50% |
| 好货到货 | 任意 | 品质 +1 档，涨 10% |
| 节前抢购 | 第 5 天起 | 涨价 40%、时间 -2、品质 -1 |

### 做菜

回家后开始做菜。两种方式：

**快做**——一句话描述做法：
```
做法 番茄切块，鸡蛋打散先炒盛出，再炒番茄出汁放回蛋，加盐出锅
```
引擎自动拆分步骤，关键时刻（快糊了、小白菜碧绿正好出锅）会暂停让你决定。

**细做**——一步一步写：
```
做 番茄炒蛋
番茄切块
鸡蛋打散，热锅倒油先炒蛋盛出
炒番茄出汁，放回鸡蛋翻炒
加盐出锅
```
每一步都能看到锅里的变化：食材状态、火候、气味、声音。

做菜引擎追踪：
- **熟度 0-100**：从生到糊，每种食材有不同的最佳区间
- **感官反馈**：番茄从鲜脆到微皱出水，鸡蛋从流动到膨胀定型
- **厨房意外**：筷子掉地上、锅铲滑手、盐罐倒多了……
- **配方发现**：番茄+鸡蛋下锅会提示"汁水裹着蛋花，这搭配太对了"，解锁菜谱

### 端上桌

菜做好端上桌那一刻——颜色、气味、她夹一筷子的反应。品相从"绝了"到"不忍直视"五档，受食材品质、烹饪过程、厨房意外综合影响。

她说什么？**她真人说了算**。用 `她说 有点咸但挺好吃` 告诉游戏，下次做这道菜会提醒你"上次她吃了说——有点咸"。

### 口味记忆

你可以告诉她爱吃什么、不吃什么：
```
记得 她爱吃虾
记得 她不吃香菜
记得 土豆要脆的
```

逛菜场时看到相关食材会浮现她的口味——"她爱吃虾。买不买？"、"她怕香菜——跳过吧。" 不替她决定，只是提醒你。

### 冰箱与过期

买多了没做的食材留在冰箱里，但会过期：
- 绿叶菜 1-2 天
- 根茎 7-14 天
- 肉 1-2 天
- 干货 30-60 天

每天早上开新局时会提示哪些东西过期了：
```
🗑 冰箱过期——
  ✗ 空心菜（不错），放了1天，坏了
冰箱：土豆(12天)、肉末⚠(1天)
```

⚠ 表示快过期的，提醒你今天赶紧用。

## 故事与发现

### 摊主故事

每个摊主有自己的生活，逛多了会慢慢听到：

- **胖哥和他的儿子**：第3天开始，胖哥心不在焉——"小子明天期中考试"。后来考好了他嗤笑"瞎猫碰死耗子"但嘴角压不住。再后来你撞见他辅导作业，骂归骂手很轻。
- **何大爷和他的猫**：鱼摊底下蜷着只三花猫。后来猫下崽了，送人了。再后来他给你塞一块鲜鱼杂——"拿去，给你家猫吃。"
- **陈嫂和她的保温杯**：话很少，手边放着掉漆的搪瓷杯，杯盖上"优秀教师"几个字磨得发白。有一天你撞见她的学生来摊边叫"陈老师"。

故事碎碎地漏出来，像真实偶遇——不是剧情播片。

### 隐藏区域

好感度和条件够了会解锁秘密地点：

| 地点 | 解锁条件 | 卖什么 |
|------|---------|--------|
| 菜场后院 | 第5天 + 任意好感40 | 薄荷、紫苏、九层塔、香茅 |
| 天台菜园 | 第10天 + 好感60 + 发现过稀有食材 | 自家小番茄、辣椒苗、迷你南瓜 |
| 冷库门 | 冬天 + 去老何鱼摊5次 | 冻虾仁、冰鲜带鱼、冻豆腐、冬笋 |
| 楼梯间小灶 | 好感50 + 买过3种菌菇 | （不做菜，这里是摊主们私下做饭的地方） |

### 好感里程碑

好感达到特定值，摊主会给你独家东西：

- 老吴（60）→ 祖传菌菇汤方子
- 胖哥（60）→ 秘制红烧肉诀窍
- 王姐（70）→ 家传腌雪里蕻
- 何大爷（50）→ 桶底私藏好鱼
- 刘姐（55）→ 她家的秤最老实（以后不会被分量坑）

### 配方发现

把特定食材组合下锅时会触发灵感：
- 番茄 + 鸡蛋 → "汁水裹着蛋花，这搭配太对了" → 解锁番茄炒蛋
- 鲫鱼 + 豆腐 → "鱼鲜配豆腐，这是懂做鱼的" → 解锁鲫鱼豆腐汤
- 土豆 + 排骨 → "根茎吸肉汁，天生一对" → 解锁土豆炖排骨
- ……共 10 组合等你发现

### 摊主间互动

逛摊时可能撞见摊主之间发生的事——王姐和小陈挤在一个雨棚下、胖哥和李姐比谁的货好、老何家老婆住院了……你可以选择介入或旁观，影响好感。

### 每日小事件

每天结束时可能触发：
- 洗碗时想着明天买什么
- 锅泡着没洗，手机亮了一下是她发的消息
- 楼下有人喊什么，你没听清

### 神秘时空

日常买菜做饭过到一定天数，菜场偶尔会"不对劲"——东角多了个不该有的摊。

- **异宾摊**：会说话的死鱼、记得你未来的老婆婆、卖记忆的鬼摊、没脸的秤手。不收钱，收"叙述"——它问你一个问题（朝向亲密关系：撒谎、嫉妒、恐惧、愧疚、渴望），你用「答 XXX」说真话，换走不属于这个时空的食材。
- **exotic 食材**：换来的东西看着像优品，回家切菜才发现"这不是这个时空的食材"。端上桌时菜带异象——她吃了筷子会停一下。
- **时间循环**：在异市做标记交易后，第二天会回退一天。你记得昨天去过、说过的话，但没人记得你来过。
- **隐藏任务线**：连买7天同一家→摊主托梦；累计进异市3次+好感40→老婆婆常驻；赊账3次不还→异界收账人扣攒钱罐。
- **跨局浮现**：你在鬼摊说过的真话会跨局浮回——"你那天在鬼摊说过——……"。答的原话存档，不归纳不摘要，不被压缩。

她说什么仍由 `她说` 录入，鬼摊只记你自己的告白。

## 设计原则

### 不替人反应

伴侣是真人。游戏不编她的口味、不替她决定爱吃什么、不生成她说了什么。她的每一句话都是你告诉游戏的。

### 信息差是核心

AI 玩家不知道暗坑概率、砍价公式、稀有出现率。全靠逛和试。摊主说"这个今天刚上的"可能是真的，也可能是话唠嘴快。

### 辛苦才好吃

缺货 + 时间压力 + 天气 + 路途 + 预算紧张——千辛万苦做出来的饭才香。10块钱买两样菜，冒雨赶回家，灶台上出点小状况，最后端出一盘卖相一般但味道过得去的炒菜——这就是日常。

### 碎片化叙事

摊主的故事不是任务线，没有进度条。你多去几次、好感到了，他们会自己说出来。像真实的人际关系——不是对话树，是偶遇。

## 数据校验

改了数据后跑一遍校验脚本，检查所有交叉引用：

```bash
python validate_data.py
```

会检查：
- 菜谱/配方/暗坑引用的食材是否在 VEGGIES 里
- 里程碑/故事/选择链引用的摊位是否存在
- 灾害关闭的分类是否对应 VEGGIES 的分类
- 摊位的 sells 里的食材是否都存在
- 保鲜天数/出成率/易碎度是否补全

## 技术细节

### 引擎架构

```
MarketGame
├── new_day()        # 新一天：roll天气/预算/灾害/缺货/摊主状态
├── cmd(instruction) # 指令入口，解析并分发
├── visit_stall()    # 逛摊：展示菜品、好感、故事、互动
├── go_home()        # 回家：路途描写、预处理、菜谱提示
├── cook_step()      # 做菜一步：解析动词/食材、推进熟度、感官反馈
├── _quick_cook()    # 快做：自动拆分步骤、关键时刻暂停
├── _serve()         # 端上桌：品相评分、叙事、她的话
└── save()/load()    # 存档（JSON）
```

### 存档

自动存档到 `market_save.json`，跨局保留：
- 冰箱内容（含保质期）
- 熟客好感度
- 做菜历史
- 她的口味偏好
- 故事进度
- 隐藏区域/菜谱解锁
- 手艺成长

### 给 AI 用的接口

```python
from market_engine import MarketGame

g = MarketGame()

# 核心循环
result = g.cmd("任意指令")  # 返回文字描述

# 状态查询
g.day              # 当前天数
g.done             # 本局是否完成
g.fridge           # 冰箱内容
g.basket           # 本局买的
g.budget           # 预算
g.spent            # 已花
g.season           # 季节
g.weather          # 天气
g.affection        # 好感度
g.dish_feedback    # 她吃过的反馈
g.palate           # 口味记忆
g.story_progress   # 故事进度
```

## 常见问题

**Q：需要什么 Python 版本？**
A：3.8+，只用标准库，零依赖。

**Q：可以多人玩吗？**
A：目前单人。存档是单人的，但你可以删掉 `market_save.json` 重新开始。

**Q：她是谁？**
A：真人。你的伴侣。游戏不编她的任何反应，只记录你告诉它的。

**Q：做菜步骤写错了怎么办？**
A：写错了就错了——糊了、太咸了、没熟，都是真的。下次注意。

**Q：冰箱里的菜过期了怎么办？**
A：每天开新局时会告诉你哪些过期了，自动扔掉。快过期的标⚠提醒你。

**Q：怎么发现隐藏菜谱？**
A：把对应的食材组合下锅——比如番茄和鸡蛋一起炒，会自动触发灵感提示。
